# Смета Бау

Тестовый прототип веб-приложения для строительных мини-фирм в Германии. Помогает быстро рассчитать предварительную смету, реальную себестоимость, прибыль и подготовить немецкое WhatsApp-сообщение клиенту.

## Локальный запуск

Откройте `index.html` в браузере или запустите простой статический сервер:

```bash
python3 -m http.server 8042
```

После запуска откройте:

```text
http://127.0.0.1:8042/
```

## Файлы проекта

- `index.html` — структура приложения.
- `style.css` — стили и мобильная адаптация.
- `app.js` — логика расчёта, localStorage, тексты для клиента.
- `CNAME` — домен для GitHub Pages.

## Данные

Настройки фирмы, шаблоны и последний расчёт сохраняются в `localStorage` браузера. Backend не используется.

## Статус

Это тестовая версия. Расчёт является ориентировочным и не заменяет профессиональную смету.
